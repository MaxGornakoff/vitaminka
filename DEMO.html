🤖 Vitaminka Assistant

Универсальный AI-ассистент для интернет-магазинов

✓ MVP Ready

🎯 Основные возможности

  • Анимированный персонаж (Rive) с интерактивными состояниями
  • Синтез речи на русском языке (Web Speech API)
  • Универсальный виджет без зависимостей (Web Components)
  • Multi-tenant архитектура (поддержка множества магазинов)
  • Shadow DOM для полной изоляции стилей
  • Работает на любом сайте (WordPress, Shopify, самописном сайте)

⚙️ Технологический стек

Backend:
  • Python 3.11 + FastAPI
  • PostgreSQL + Redis
  • LangChain + ChromaDB
  • OpenAI GPT-4
Frontend:
  • Vanilla JavaScript
  • Web Components
  • Rive Animation
  • Web Speech API

🚀 Быстрый старт (локальная разработка)

1. Клонируйте репозиторий и перейдите в директорию:

git clone https://github.com/yourusername/vitaminka
cd vitaminka

2. Запустите Docker Compose:

docker-compose up -d

3. Проверьте статус:

curl http://localhost:8000/api/health

4. Откройте в браузере:

  • API Docs: http://localhost:8000/docs
  • ReDoc: http://localhost:8000/redoc

📱 Интеграция на магазине

Добавьте перед закрытием </body>:

<script>
  window.VITAMINKA_API_URL = 'https://your-api.com';
  window.VITAMINKA_SHOP_ID = 'your_shop_id';
  window.VITAMINKA_RIVE_FILE = 'https://cdn.com/character.riv';
</script>
<script src="https://your-api.com/widget.js"></script>

✨ Готово! Виджет появится в нижнем правом углу сайта.

👩‍🦰 Анимированный персонаж

Персонаж меняет свою анимацию в зависимости от состояния диалога:

😴 Idle
Ожидание, фоновая анимация
👂 Listening
Слушает пользователя
🤔 Thinking
Обрабатывает запрос
💬 Speaking
Отвечает с синтезом речи

📚 Читать документацию Rive

🔊 Синтез речи (Text-to-Speech)

  • Встроен в виджет, никаких дополнительных API не нужно
  • Язык: Русский (ru-RU)
  • Пользователь может включить/отключить звук кнопкой 🔊
  • Поддержка: Chrome, Firefox, Safari, Edge
